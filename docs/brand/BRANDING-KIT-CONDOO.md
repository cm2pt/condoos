# Condoo Branding Kit (Documento Vivo)

## 1) Essencia da Marca
- Nome de marca: `Condoo`
- Categoria: plataforma digital para gestao de condominios.
- Posicionamento: simples para o utilizador final, robusta para operacao profissional.
- Personalidade: clara, humana, confiavel, organizada.

## 2) Proposta de Valor (Copy Base)
- Headline: `Gestao de condominio simples, clara e preparada para crescer.`
- Supporting line: `Reune quotas, ocorrencias, assembleias e documentos num unico espaco para administracao e condominos.`

## 3) Sistema Visual
- Simbolo (icone): `/Users/luislaginha/Library/CloudStorage/Dropbox/Condoos/public/brand/condoo-symbol.svg`
- Wordmark (logo horizontal): `/Users/luislaginha/Library/CloudStorage/Dropbox/Condoos/public/brand/condoo-wordmark.svg`
- Uso recomendado:
  - `wordmark` em login, sidebar e materiais institucionais.
  - `symbol` em favicon, badges, cabecalhos compactos e watermarks.

## 4) Cores Oficiais
- `--brand-deep`: `#145A52` (confiança e estabilidade)
- `--brand-mid`: `#1F7A6E` (acao e progresso)
- `--brand-sand`: `#F0E5CF` (calor e proximidade)
- `--brand-highlight`: `#F6C680` (destaque pontual)

## 5) Tipografia
- UI base: `"Avenir Next", "Segoe UI", "Helvetica Neue", sans-serif`
- Titulos/accent: `"Trebuchet MS", "Gill Sans", sans-serif`
- Regra: contraste elevado, textos curtos, sem jargao tecnico para interfaces publicas.

## 6) Componentes de Marca
- `brand-lockup`: logo + tagline no sidebar.
- `workspace-brandline`: simbolo + contexto no cabecalho de modulo.
- `login-brand`: wordmark hero do login.
- `login-card-brand`: simbolo no cartao de autenticacao.
- watermark global com `condoo-symbol.svg` no canto inferior.

## 7) Diretrizes de UX de Marca
- Linguagem para cliente final: beneficio primeiro, tecnologia em segundo plano.
- Evitar termos tecnicos em superficies publicas (ex: `RBAC`, `API`) salvo contexto interno.
- Estados de demo devem estar explicitamente marcados como temporarios.

## 8) Aplicacao no Produto (Estado Atual)
- Login orientado a cliente final e com area de demo identificada.
- Marca aplicada no login, sidebar, header e watermark global.
- Assets prontos para reaproveitamento em propostas comerciais e website.
