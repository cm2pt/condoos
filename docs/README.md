# Plataforma de Gestao de Condominios (Portugal) - Base Inicial

Este diretorio contem a base documental para arrancar desenvolvimento da V1.

## Estrutura

- `docs/product/PRD-V1.md`: especificacao funcional da V1.
- `docs/product/BACKLOG-V1.md`: backlog por sprint com criterios de aceitacao.
- `docs/tech/DATA_MODEL.md`: modelo de dados e relacoes principais.
- `docs/FRONTEND-PROTOTYPE.md`: estado atual da interface e testes de acoes rapidas.
- `docs/templates/legal/`: templates legais e administrativos (rascunho).
- `docs/templates/operations/`: templates operacionais de gestao.
- `data/synthetic/`: dataset sintetico para desenvolvimento e QA.

## Nota importante

Todos os templates legais deste repositorio sao **rascunhos tecnicos** para acelerar a implementacao de produto.
Devem ser validados por apoio juridico antes de uso comercial.
