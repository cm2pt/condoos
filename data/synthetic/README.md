# Dados sinteticos para desenvolvimento

Ficheiro principal:

- `data/synthetic/condominio_portugal_seed.json`

## Cobertura

- 1 condominio em Portugal (Lisboa).
- 11 andares.
- 30 fracoes (habitacao, loja, estacionamento, arrecadacao).
- Pessoas com papeis de proprietario, arrendatario, gestor e fornecedor.
- Encargos e pagamentos (inclui pagos, em aberto, parcial e atraso).
- Documentos cobrindo categorias administrativas, financeiras e compliance.
- Ocorrencias com diferentes estados, prioridades e fornecedores.
- Assembleias com pontos de votacao e resumo de resultados.

## Uso recomendado

- Importar em ambiente de desenvolvimento para testes E2E.
- Usar como base para seeds de base de dados ou mocks de API.
- Nao usar em ambiente de producao.
